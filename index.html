WP Animal ONG

WPAnimal ONG é um plugin para WordPress que visa auxiliar na gestão de ONG's que resgatam e cuidam de animais.

Funcionalidades Versao 1.0:

  • Cadastro de Apoiadores
  • Financeiro
  • * Caixas
  • * Lancamentos
  • Cadastro de Animais
  • Saude
  • * Medicamentos
  • * Entrada de Medicamentos
  • * Aplicacao de Medicamentos
  • * Veterinarios
  • * Procedimentos
  • Adocoes
  • Usuarios
  • Relatorios

Instalacao

Extraia a pasta wp-animal-ong pra dentro do seu /wp-content/plugins e ative

$ git clone https://github.com/joelgarciajr84/wp-animal-ong.git wp-animal-ong

Quer Contribuir?

  1. Faça um fork do repositório;
  2. Configure seu fork;
  3. Verifique as issues e escolha uma que não tenha ninguém trabalhando;
  4. Sincronize seu fork;
  5. Crie uma branch para trabalhar na issue da sua responsabilidade: git checkout -b issue-17;
  6. Commit as alterações que você fez: git commit -m 'Add Alguma coisa nova';
  7. Faça um push para a branch: git push origin issue-17;
  8. Faça um Pull Request :D

Observação: Se deseja contribuir com algo que não foi registrado nas issues é importante criar e assinar para evitar que outra pessoa comece a trabalhar na mesma coisa que você.

Se precisar de ajuda para executar algum dos procedimentos acima acesse o link e aprenda como fazer um Pull Request.
